This demo shows you how to setup and use RTL both in editor and in code. RTL in used with both Unity UI and old GUI systems.

To use RTL in design mode, easily add the RTLText component to any UI Text element and press convert button in Inspector. If you need to use RTL in code, you can call RTLService.Convert() method.
